Comment ça marche
1
Partage ton code
Envoie ton code unique à tes amis par message, courriel ou via les boutons de partage.
2
Ton ami s'inscrit
Il entre ton code lors de son inscription ou dans cette page. Le parrainage est enregistré.

4
Vous êtes récompensés
Tu reçois +100 Points Impact et +50 XP. Lui reçoit aussi un bonus de bienvenue.

🛡️ Programme anti-fraude

  • La récompense n'est jamais attribuée à l'inscription — seulement après une action réelle et vérifiée.
  • Un seul code de parrainage peut être utilisé par personne.
  • Tu ne peux pas utiliser ton propre code de parrainage.
  • La détection de comptes multiples entraîne l'annulation de toutes les récompenses et la suspension du compte.
  • Les récompenses suspectes peuvent être annulées par l'administration à tout moment.
  • Le parrainage est limité à des actions vérifiables sur la plateforme (livraisons, covoiturages).
