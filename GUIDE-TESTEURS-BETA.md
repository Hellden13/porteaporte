# Guide simple pour les testeurs beta

Merci de tester PorteaPorte.

La plateforme est en beta locale. Le but est de trouver ce qui est clair, ce qui bloque et ce qui doit etre simplifie avant une ouverture plus large.

## Regles de test

- Utilise seulement de petits colis.
- Ne mets pas d'objet fragile ou de grande valeur.
- Ne partage pas d'information sensible inutile.
- Signale tout comportement bizarre.
- Ne teste pas plusieurs paiements identiques si un paiement bloque.
- Prends une capture d'ecran si possible.

## Quoi tester

### Si tu es expediteur

- creer un compte;
- confirmer ton email;
- creer une livraison simple;
- verifier si le prix est clair;
- payer;
- verifier si la livraison apparait dans ton dashboard;
- suivre le statut;
- verifier si tu comprends la confirmation destinataire.

### Si tu es livreur

- creer un compte;
- confirmer ton email;
- remplir la verification;
- verifier si ton statut est clair;
- regarder les missions disponibles seulement si tu es verifie;
- accepter une mission test;
- verifier les details;
- marquer la livraison comme livree seulement quand c'est vrai;
- deposer une preuve si livraison sans humain.

### Si tu es destinataire

- recevoir le colis;
- recevoir ou obtenir le code;
- confirmer la reception;
- signaler si le code, la page ou le message n'est pas clair.

## Quoi noter

Pour chaque probleme, note:

- page;
- bouton clique;
- ce que tu voulais faire;
- ce qui est arrive;
- message d'erreur exact;
- heure approximative;
- appareil utilise: iPhone, Android, ordinateur;
- navigateur: Chrome, Safari, Edge;
- capture d'ecran si possible.

## Exemples utiles

Bon signalement:

"Sur iPhone, page paiement, apres avoir clique Payer, le bouton tourne mais rien ne se passe. Il etait environ 14h10. Livraison: lettre Quebec vers Levis."

Signalement moins utile:

"Ca marche pas."

## Ce qu'on veut apprendre

- Est-ce simple?
- Est-ce rassurant?
- Est-ce que les boutons sont clairs?
- Est-ce que le paiement inspire confiance?
- Est-ce que le statut de livraison est facile a comprendre?
- Est-ce que le livreur sait quoi faire?
- Est-ce que le destinataire comprend comment confirmer?
