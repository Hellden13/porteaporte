## ✅ Comment ça marche après upload

Le code détecte automatiquement la présence des photos :
- **Si présente** → affichée
- **Si absente** → fallback emoji/gradient gardé (zéro casse)

Tu peux donc uploader **par lots** ou **une à la fois** — le site reste fonctionnel à tout moment.
