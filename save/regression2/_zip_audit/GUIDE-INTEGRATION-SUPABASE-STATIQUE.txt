🗄️ CRÉER LES TABLES SUPABASE
═══════════════════════════════════════════════════════════════════════════════

Si les tables n'existent pas, les créer dans Supabase:

URL Supabase: https://supabase.com/dashboard/project/miqrircrfpzkmvvacgwt/sql

CRÉER: profiles (utilisateurs)
━━━━━━━━━━━━━━━━━━━━━━━━━━━
CREATE TABLE profiles (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  email TEXT UNIQUE NOT NULL,
  prenom TEXT,
  nom TEXT,
  role TEXT DEFAULT 'expediteur', -- 'expediteur' ou 'livreur'
  actif BOOLEAN DEFAULT true,
  score NUMERIC DEFAULT 0, -- Note 0-5
  coins INTEGER DEFAULT 0, -- PorteCoins
  livraisons INTEGER DEFAULT 0,
  cree_le TIMESTAMP DEFAULT NOW(),
  mis_a_jour_le TIMESTAMP DEFAULT NOW()
);

CRÉER: missions (livraisons)
━━━━━━━━━━━━━━━━━━━━━━━━━━
CREATE TABLE missions (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  titre TEXT,
  description TEXT,
  depart TEXT,
  arrivee TEXT,
  date TIMESTAMP,
  prix NUMERIC DEFAULT 0,
  statut TEXT DEFAULT 'en_attente', -- 'en_attente', 'en_route', 'livree'
  expediteur_id UUID REFERENCES profiles(id),
  livreur_id UUID REFERENCES profiles(id),
  cree_le TIMESTAMP DEFAULT NOW()
);

CRÉER: payment_transactions (paiements)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
CREATE TABLE payment_transactions (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  delivery_id UUID REFERENCES missions(id),
  total_amount NUMERIC NOT NULL,
  livreur_amount NUMERIC,
  porteaporte_amount NUMERIC,
  causes_amount NUMERIC,
  concours_amount NUMERIC,
  claude_amount NUMERIC,
  fees_amount NUMERIC,
  status TEXT DEFAULT 'pending', -- 'pending', 'released', 'refunded'
  created_at TIMESTAMP DEFAULT NOW()
);

📊 FONCTIONNALITÉS DISPONIBLES
═══════════════════════════════════════════════════════════════════════════════

✅ UTILISATEURS (profiles):
  • adminCRUD.loadUsers() → Affiche tous les utilisateurs
  • adminCRUD.createUser(data) → Créer un utilisateur
  • adminCRUD.updateUser(id, data) → Modifier un utilisateur
  • adminCRUD.deleteUser(id) → Supprimer un utilisateur

✅ LIVRAISONS (missions):
  • adminCRUD.loadDeliveries() → Affiche toutes les livraisons
  • adminCRUD.createDelivery(data) → Créer une livraison
  • adminCRUD.updateDelivery(id, data) → Modifier une livraison
  • adminCRUD.deleteDelivery(id) → Supprimer une livraison

✅ LIVREURS (profiles filtrés):
  • adminCRUD.loadDrivers() → Affiche les livreurs (role='livreur')

✅ TRANSACTIONS (payment_transactions):
  • adminCRUD.loadTransactions() → Affiche les transactions
