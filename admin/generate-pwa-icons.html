🎨 Générateur d'icônes PWA

Génère les fichiers icon-192.png et icon-512.png à partir du logo SVG de PorteàPorte. À faire une seule fois, puis enregistre les fichiers dans le dépôt.

  1. Clique "Générer les icônes" ci-dessous.
  2. Clique "⬇️ Télécharger icon-192.png" → enregistre dans Site/icons/icon-192.png
  3. Clique "⬇️ Télécharger icon-512.png" → enregistre dans Site/icons/icon-512.png
  4. Dans VS Code / terminal : git add icons/ && git commit -m "feat: icones PWA PNG" && git push
192 × 192 px (Android, Chrome)
512 × 512 px (Splash screen, Play Store)
⚠️ iOS (Safari) — Apple requiert une icône PNG pour afficher l'icône sur l'écran d'accueil. Une fois les fichiers dans icons/ et le push fait, les utilisateurs iOS verront l'icône correctement quand ils ajoutent l'app à l'écran d'accueil.
