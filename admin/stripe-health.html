🛠️ Si un test échoue

STRIPE_SECRET_KEY manquante → va dans Vercel Settings → Environment Variables, ajoute STRIPE_SECRET_KEY = ta clé (commence par sk_live_ en prod), puis Redeploy.

STRIPE_WEBHOOK_SECRET manquante → va dans Stripe Dashboard → Webhooks, clique sur ton webhook (ou crée-le pointant sur https://porteaporte.site/api/stripe-webhook), copie le "Signing secret" (commence par whsec_) et mets-le dans Vercel.

Aucun webhook URL Stripe → Stripe Dashboard → Webhooks → "Add endpoint" → URL = https://porteaporte.site/api/stripe-webhook → Events à activer : payment_intent.succeeded, payment_intent.amount_capturable_updated, payment_intent.payment_failed, payout.paid, charge.refunded.

Webhook events 24h = 0 → soit pas de transactions hier (normal au début), soit le webhook ne marche pas. Va dans Stripe Dashboard → ton webhook → "Recent attempts" pour voir le code HTTP retour. Si 401 → ton WEBHOOK_SECRET est mal collé. Si 500 → bug dans /api/stripe-webhook.js.
