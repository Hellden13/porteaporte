✨ Toutes nos fonctionnalités

Sécurité maximale, matching IA, transparence totale. Une plateforme pensée pour la confiance entre expéditeurs, livreurs et destinataires.

🔒 Sécurité & Identité

🆔

Carte d'identité numérique NEW

Chaque livreur a une carte virtuelle avec photo + QR code à scanner. L'expéditeur vérifie l'identité avant d'ouvrir.

🤖

IA Face matching NEW

Au pickup, la selfie du livreur est comparée automatiquement à sa selfie KYC. Si différent → alerte.

⚡

Vérification instantanée Stripe NEW

KYC en 30 secondes via Stripe Identity (permis + selfie + liveness). Plus besoin d'attendre 48h.

🤳

Selfie + GPS au pickup NEW

Le livreur doit prendre une selfie au moment où il récupère le colis. GPS + photo = preuve d'identité incontestable.

🔑

Double code (pickup + réception)

Un code pour le livreur à l'arrivée + un code pour le destinataire. Personne ne peut détourner le colis.

🛰️ Tracking & GPS

📍

GPS obligatoire NEW

Le livreur s'engage à laisser son GPS activé jusqu'à la livraison. S'il le désactive → alerte automatique.

🗺️

Tracking live avec ETA NEW

L'expéditeur et le destinataire voient la position du livreur en temps réel + ETA + progress bar.

📲

Notifications par email

À chaque étape (création, paiement, départ, arrivée, confirmation), tous les acteurs sont prévenus.

🤖 Matching IA

🧠

Algo Haversine NEW

Distance GPS précise entre 44 villes du Québec. Matching intelligent selon transport, taille et trajet.

🚗

Covoiturage de colis NEW

Le livreur configure sa route (origine, destination, déviation km). L'IA trouve les missions sur son chemin.

📦

Matching taille colis NEW

Un vélo ne livre pas un frigidaire. L'algo filtre les missions selon ton véhicule.

💬 Confiance & Communication

⚖️

Signalement bidirectionnel NEW

Expéditeur, livreur et destinataire peuvent se signaler mutuellement. 48h pour contester. Admin tranche.

⭐

Score de fiabilité NEW

Chaque utilisateur a un score 0-100 basé sur les manquements validés. Visible publiquement.

💬

Messagerie groupe 3-4 personnes NEW

Canal partagé entre expéditeur, livreur, destinataire et admin si besoin.

🚨

Intelligence d'adresses NEW

Les livreurs partagent des notes : chien dangereux, code porte, étages... Communauté solidaire.

🚨 Gestion des imprévus

⏱️

Pré-confirmation XL NEW

Pour les gros colis (électroménager), le destinataire doit confirmer sa présence avant départ du livreur (15 min).

↩️

3 actions imprévu

Dépôt sécurisé (photo), Re-livraison programmée, ou Retour expéditeur — selon la situation.

💰

Compensation automatique NEW

Si le destinataire est fautif (absent, refus), le livreur reçoit 25%-50% de compensation pour son déplacement.

💳 Paiement sécurisé

🏦

Escrow Stripe

Le paiement est bloqué dès la création. Libéré au livreur seulement après confirmation du destinataire.

⚙️

Connect Express pour livreurs

Les livreurs reçoivent leurs gains sur leur compte bancaire via Stripe Connect.
