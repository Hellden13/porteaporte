🔍 Comment ça marche

  1. 5% de chaque livraison est automatiquement mis de côté dans un fonds dédié
  2. Chaque trimestre, notre communauté vote pour répartir les fonds entre les organismes partenaires
  3. Versements transparents : tous les paiements sont publiés ici avec les preuves
  4. Audit annuel par un comptable indépendant québécois
